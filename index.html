Q-interactive FAQ

Stel een vraag over Q-interactive

Veelgestelde vragen

  • Wat is Q-interactive?

    Q-interactive is een digitaal platform voor het afnemen van psychologische tests via iPads.

  • Welke tests zijn beschikbaar op Q-interactive?

    Er zijn verschillende tests beschikbaar, waaronder WISC-V-NL, WAIS-IV-NL, WPPSI-IV-NL, WMS-IV-NL, CELF-5-NL, en meer.

  • Hoe werkt de licentie voor Q-interactive?

    Er zijn jaarlicenties beschikbaar die toegang geven tot alle tests op het platform. Er is ook een speciale Taal&Spraak licentie voor logopedisten.
